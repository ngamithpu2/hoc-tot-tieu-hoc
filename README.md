# Học Tốt Tiểu Học

Website bán khóa học và luyện tập dành cho học sinh tiểu học, xây dựng bằng Next.js và sẵn sàng triển khai trên Vercel.

## Chạy trên máy

```bash
npm install
npm run dev
```

Mở `http://localhost:3000`.

## Triển khai

Nhập repository này vào Vercel. Framework Preset chọn **Next.js** và giữ nguyên các thiết lập build mặc định.
