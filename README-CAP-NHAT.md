# Cập nhật Tiếng Việt 4 · Tập 1 — giao diện v2

Gói này chỉ cập nhật riêng khóa `tieng-viet-4-kntt-tap-1`.

## Cài đặt

Sao chép toàn bộ thư mục `data` và `public` vào thư mục gốc của dự án, cho phép ghi đè các tệp trùng tên. Không xóa các thư mục hoặc khóa học khác.

Đường dẫn sau khi cập nhật:

`/lop-4/khoa-hoc/tieng-viet-4-kntt/tap-1`

## Kiểm tra

```bash
npm run course:validate -- tieng-viet-4-kntt-tap-1 v2
npm run lint
npm run build
```

## Hoàn tác về v1

```bash
npm run course:activate -- tieng-viet-4-kntt-tap-1 v1
```

Nội dung học tập của v2 giữ nguyên so với v1: 32 bài học và 869 câu hỏi.
